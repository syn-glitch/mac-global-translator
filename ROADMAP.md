# 🚀 향후 개발 로드맵 (Roadmap)

Mac Global Translator를 더욱 강력한 어학 학습 도구로 발전시키기 위한 추가 개발 계획입니다.

## 1. 🗣️ 고품질 원어민 음성 (High-Quality TTS)
현재 Mac 기본 TTS(NSSpeechSynthesizer)의 기계적인 발음을 개선하기 위해, **AI 기반 음성 합성 엔진**을 도입합니다.

### 🌟 OpenAI (ChatGPT) TTS
- **특징**: `alloy`, `echo`, `shimmer` 등 매우 자연스럽고 감정 표현이 가능한 목소리.
- **장점**: 가격이 저렴하고(1,000자당 약 $0.015), 응답 속도가 빠름.
- **구현 계획**:
    - OpenAI API Key 입력란 추가 (`UserDefaults` / `Keychain` 저장)
    - `TranslateService`와 별도로 `TTSService` 모듈 분리
    - 팝업의 [🔊 듣기] 버튼 클릭 시 MP3 스트리밍 재생

### 🌟 ElevenLabs API
- **특징**: 현존 최고 수준의 자연스러움, 다양한 캐릭터 목소리(예: 주디 홉스 스타일) 커스텀 가능.
- **단점**: OpenAI보다 비용이 높음.

---

## 2. 📚 나만의 단어장 (Vocabulary Builder)
번역한 문장을 단순히 보고 넘기는 것이 아니라, **학습용 데이터**로 축적합니다.

- **[⭐ 저장] 버튼 추가**: 팝업창에 즐겨찾기 버튼 생성
- **데이터 활용**:
    - `CSV` 또는 `JSON` 파일로 내보내기 (Anki 등 암기 앱 연동)
    - Apple '메모(Notes)' 앱 또는 '미리알림(Reminders)'에 자동 추가

## 3. 💡 AI 문법/표현 해설 (One-Point Lesson)
단순 번역을 넘어, **왜 이렇게 해석되는지** 알려주는 과외 선생님 기능입니다.

- **프롬프트 개선**: 번역 요청 시 "주요 단어 3개와 문법적 특징을 간단히 요약해줘"라고 추가 요청.
- **UI 개선**: 번역 결과 하단에 `[문법 해설]` 섹션 추가 (토글 형태).

---

## 4. ⌨️ 단축키 지원 (Global Hotkey)
- 마우스 드래그 외에 `Opt + Space` 등 단축키로 번역 팝업 호출.
- 입력창을 띄워 직접 타이핑해서 번역하는 모드 지원.
